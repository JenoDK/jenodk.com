#### **Bracket Selection** ([GitHub](https://github.com/JenoDK/bracket-selection))

This was a feature I frequently used while working with Eclipse IDE and missed when transitioning to IntelliJ long ago. Meanwhile IntelliJ has a keybind to replace some of the functionality of this plugin.

#### **jdump** ([GitHub](https://github.com/JenoDK/jdump))
A small CLI written in Python to easily handle SQL dumps across different projects. I wrote this pretty early on in my career as I kept forgetting the commands and my dumps were all over the place.

#### **JBehave Support** ([GitHub](https://github.com/witspirit/IntelliJBehave)) - Maintainer
A widely used plugin for IntelliJ IDE with 160,000 total downloads. While working at Agilent, I used this plugin in our project and noticed some issues. I created a ticket and submitted a PR to fix it. The PR was approved and merged, but no new version was released, so I contacted the repository owner to offer assistance. This led to me becoming a maintainer with the ability to approve PRs and release new versions.

#### **Strava Stats** ([GitHub](https://github.com/JenoDK/strava-stats-jeno)) - Work in Progress
I use Strava extensively but found it difficult to locate past activities, so I decided to build my own solution. Using the [Strava API](https://developers.strava.com/docs/reference/) and a React frontend, the app provides an enhanced way of filtering through activities. It includes support for location-based search, allowing users to find activities within a specific radius of a location on the map (e.g., activities within 5km of a vacation address in Italy).
